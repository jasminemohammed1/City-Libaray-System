# 📚 City Library System

A **City Library Management System** developed using **C# and Object-Oriented Programming principles**.  
The system simulates the core operations of a city library, including managing books, users, borrowing processes, and librarians.

This project demonstrates the practical use of **OOP concepts such as inheritance, encapsulation, abstraction, and modular design** in building a simple yet structured library management solution.

> ⚠️ **Note:** This project is also implemented as a **Class Library**, which can be referenced in other C# projects. You can include this library in your code using the repository:  
> [City-Library-System Class Library](https://github.com/jasminemohammed1/City_Library_System_Class_Library.git)
---

## 📖 Overview

The **City Library System** is designed to manage and organize library operations efficiently.  
It provides a structured model for handling:

- Library users
- Librarians
- Books
- Borrowing operations
- Library records

The system focuses on **clean class design and object relationships**, making it suitable as a **learning project for Object-Oriented Programming and C# application development**.

---

## ✨ Features

- 📚 **Book Management**  
  - Add and manage book records  
  - Store book details such as title, author, and availability

- 👤 **User Management**  
  - Support different types of users  
  - Maintain information about library members

- 🧑‍💼 **Librarian Management**  
  - Dedicated librarian class responsible for managing the system  
  - Control over book operations and library activities

- 🔄 **Borrowing System**  
  - Track borrowed books  
  - Manage book borrowing operations

- 🧩 **Object-Oriented Architecture**  
  - Clean class hierarchy  
  - Reusable and modular design

---

## 🏗️ System Architecture

The project follows an **Object-Oriented Design** where the system is divided into multiple models representing real-world entities.

## ⚙️ Technologies Used

- **Language:** C#  
- **Framework:** .NET  
- **Programming Paradigm:** Object-Oriented Programming (OOP)  
- **Development Environment:** Visual Studio

## 🧠 OOP Concepts Applied

This project demonstrates the use of important **Object-Oriented Programming principles**:

| Concept        | Implementation                                 |
|----------------|-----------------------------------------------|
| Encapsulation  | Private properties with controlled access     |
| Inheritance    | `Librarian` and `Member` extend `LibraryUser` |
| Abstraction    | Classes represent real-world entities        |
| Modularity     | Separation of models and logic               |
